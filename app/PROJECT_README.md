# KneeGuard Flutter Application

Aplikacja Flutter do monitorowania i ćwiczenia zdrowia kolan.

## Wymagania

- Flutter SDK (3.0.0 lub nowsze)
- Dart 3.0.0 lub nowsze
- Android Studio / Xcode (dla emulatorów)

## Instalacja

1. Upewnij się, że masz zainstalowany Flutter SDK
2. Przejdź do katalogu projektu
3. Uruchom: `flutter pub get`

## Uruchamianie aplikacji

```bash
# Uruchomienie na emulatorze/urządzeniu
flutter run

# Uruchomienie w trybie release
flutter run --release

# Uruchomienie na sieci Web
flutter run -d chrome

# Uruchomienie na Windowsie
flutter run -d windows
```

## Struktura projektu

```
├── lib/              # Kod źródłowy aplikacji
├── test/             # Testy jednostkowe
├── assets/           # Zasoby (obrazy, czcionki)
├── pubspec.yaml      # Konfiguracja projektu
└── README.md         # Ten plik
```

## Funkcje

- Monitorowanie zdrowia kolan
- Lista ćwiczeń
- Historia postępów
- Ustawienia aplikacji
